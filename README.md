## Description

This Project for test NestJS with PrismaORM, PostgresSQL and Google Translate API

## Note*

Update .env file first !!!

## Installation

```bash
$ npm install
$ npx prisma generate
$ npx prisma db push
// this command for seed tester user data
$ npx prisma db seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
